# EIEntropy
EIEntropy implements entropy-based methods for cross-level inference and data integration under moment consistency constraints. Two methods are available, Generalized Cross Entropy (GCE) and Generalized Maximum Entropy (GME). The package is available on CRAN and provides robust estimates for economics, demography, and social sciences. 
## Applications
The package is particularly suitable for:
- Small Area Estimation
- Regional and spatial economics
- Demographic analysis
- Social sciences research
- Data integration problems with limited microdata availability
## Requirements
R ≥ 3.5.0
No compilation required
